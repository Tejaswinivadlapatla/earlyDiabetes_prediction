# Diabetes Prediction using Machine Learning
## Project Overview
This project predicts whether a patient is diabetic or non-diabetic using Machine Learning techniques. The model is trained on the Pima Indians Diabetes Dataset, which contains several medical attributes such as glucose level, blood pressure, BMI, insulin level, age, and pregnancies.
The goal of this project is to analyze patient health data and build a predictive model that can assist in the early detection of diabetes.
## Dataset Information
This project uses the Pima Indians Diabetes Dataset, a well-known dataset for diabetes prediction and binary classification problems.
### Dataset Details
[Pima Indians Diabetes Dataset](https://www.kaggle.com/datasets/uciml/pima-indians-diabetes-database)
Total Records: 768
Features: 8 Input Features + 1 Target Variable
Target Variable: Outcome
0 → Non-Diabetic
1 → Diabetic

## Conclusion
This project demonstrates the application of Machine Learning in healthcare for predicting diabetes based on patient medical data. Through data preprocessing, exploratory data analysis, visualization, and Logistic Regression modeling, the system successfully classifies patients as diabetic or non-diabetic.
The project highlights the importance of data-driven decision-making in the medical field and shows how machine learning can assist in the early detection of diseases. The results indicate that features such as glucose level, BMI, insulin, and age play a significant role in diabetes prediction.
Overall, this project provides a strong foundation for developing more advanced healthcare prediction systems and can be further enhanced by using different machine learning algorithms, hyperparameter tuning, and deployment as a web application.
